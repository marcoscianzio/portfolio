Discord es una aplicación de chat, similar a programas como Skype, TeamSpeak o plataformas de comunicación profesionales como Slack. En busca de nuevos retos, Agustin Ribotta y yo decidimos clonar esta popular app. El se encargaría del Frontend y yo de la API.

## Tecnologías

- Typescript

### Backend

- Node
- Prisma
- Apollo GraphQL
- Express
- Nexus
- GraphQL-WS
- Redis

### Base de datos

- MySQL

### Frontend

- Next.js
- Tailwind
