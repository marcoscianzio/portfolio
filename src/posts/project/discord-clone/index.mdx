Discord is a free voice, video, and text chat app that's used by tens of millions of people ages 13+ to talk and hang out with their communities and friends. Looking for new challenges, Agustin Ribotta and me decided to clone this popular app.

## Technologies

- Typescript

### Backend

- Node
- Prisma
- Apollo GraphQL
- Express
- Nexus
- GraphQL-WS
- Redis

### Base de datos

- MySQL

### Frontend

- Next.js
- Tailwind
