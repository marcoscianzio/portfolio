​YouTube es un sitio Web que permite a sus usuarios subir vídeos para que otros puedan consumirlos en cualquier momento y de manera Online

La subida de videos me parecio un desafío interesante, razón por la cual decidí crear un clon de este sitio. Ya habia usado Cloudinary en un par de proyectos para las imagenes, pero no tenía idea que podría hacer lo mismo con videos.

## Technologies

- Typescript

### Backend

- Node
- Prisma
- TypeGraphQL
- Express
- Apollo GraphQL
- Redis

### Base de datos

- MySQL

### Frontend

- Next.js
- Apollo Client
- Chakra UI
- Cloudinary
