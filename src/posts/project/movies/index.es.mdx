La mayoría de apps de peliculas tienden a utilizar _The Movie Database (TMDB) API_. En esta ocasión, quería probar algo diferente. Basicamente, quería hacer mi propia API de películas desde cero.

No solo ser capaz de consultar películas, sino que también su cast y géneros. También puedo consultar actores de películas. Además, obviamente consultar por las películas de un actor.

Actualmente, existen mutaciones para crear todas las entidades, así que probablemente en el futuro convierta esta app en un dashboard.

## Tecnologías

- Typescript

### Backend

- Node
- TypeORM
- Apollo GraphQL
- TypeGraphQL
- Express
- Redis

### Base de datos

- MySQL

### Frontend

- Next.js
- Apollo Client
- Chakra UI
