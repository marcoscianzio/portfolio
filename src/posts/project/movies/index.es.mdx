La mayoría de las aplicaciones relacionadas con el cine suelen aprovechar la API de The Movie Database (TMDB). Sin embargo, en esta ocasión, decidí tomar un camino menos transitado y embarcarme en la aventura de crear mi propia API de películas desde cero.

El objetivo no se limitaba simplemente a ser capaz de consultar películas. Quería ir más allá e incorporar funcionalidades que permitieran buscar también el elenco y los géneros de cada película. No solo eso, sino que también deseaba incorporar la capacidad de buscar actores específicos y las películas en las que han trabajado. Asimismo, consideré importante que la API fuera capaz de realizar una búsqueda inversa, es decir, poder consultar las películas en las que un actor ha participado.

En su estado actual, la API ya cuenta con las mutaciones necesarias para crear todas las entidades mencionadas anteriormente. Sin embargo, mi visión va más allá. En el futuro, planeo convertir esta aplicación en un completo panel de control para la gestión de datos de películas, actores y géneros. De esta manera, no solo podríamos consultar información, sino que también podríamos interactuar activamente con ella, creando una experiencia mucho más dinámica e interesante.

## Tecnologías

- Typescript

### Backend

- Node
- TypeORM
- Apollo GraphQL
- TypeGraphQL
- Express
- Redis

### Base de datos

- MySQL

### Frontend

- Next.js
- Apollo Client
- Chakra UI
