Most movies apps tends to use _The Movie Database (TMDB) API_. In this occasion, i wanted to try something different. Basically, i wanted to make my own movie API from scratch.

Not only query movies, but also its cast and genres. Moreover, query actor's movies.

Currently, there are mutations to create all entities, so i will probably turn it into a dashboard in the future.

## Technologies

- Typescript

### Backend

- Node
- TypeORM
- Apollo GraphQL
- TypeGraphQL
- Express
- Redis

### Database

- MySQL

### Frontend

- Next.js
- Apollo Client
- Chakra UI
